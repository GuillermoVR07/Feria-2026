# Oral Lesion Triage Model

Expected checkpoint path for Hugging Face:

```text
models/oral-lesion-triage-cnn/1.0.0/model.keras
```

The technical checkpoint generated by `scripts/create_technical_checkpoint.py` is only for integration testing. It is not clinically trained, not clinically validated, and must not be used as medical evidence.

For a real model, replace `model.keras` with a checkpoint trained and validated on an appropriate oral lesion dataset using the same 3 output classes:

- `low`
- `moderate`
- `high`
